= 查看集群的运行状况和详细信息
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/use/


[role="lead"]
将 Kubernetes 集群添加到 Astra Control 后，您可以查看有关集群的详细信息，例如集群的位置，工作节点，永久性卷和存储类。

.步骤
. 选择 * 集群 * 。
. 选择集群名称。
. 查看 * 概述 * 和 * 存储 * 选项卡中的信息以查找所需的信息。
+
** * 概述 * ：有关工作节点的详细信息，包括其状态。
** * 存储 * ：与集群关联的永久性卷，包括存储类和状态。
** * 活动 * ：与集群相关的 Astra 活动。
