= 查看计算的运行状况和详细信息
:hardbreaks:
:icons: font
:imagesdir: ../media/use/


[role="lead"]
将 Kubernetes 计算添加到 Astra Control 后，您可以查看有关该计算的详细信息，例如其位置，工作节点，永久性卷和存储类。

.步骤
. 单击 * 计算 * 。
. 单击计算名称。
. 查看 * 概述 * 和 * 存储 * 选项卡中的信息以查找所需的信息。
+
** * 概述 * ：有关工作节点的详细信息，包括其状态。
** * 存储 * ：与计算关联的永久性卷，包括存储类和状态。
** * 活动 * ：与计算相关的 Astra 活动。
